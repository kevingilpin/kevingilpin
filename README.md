Hi 👋 My name is Kevin Gilpin
=============================

Frontend Developer
------------------

I primarily code on the frontend, but try to engage with my first love, the backend, as well. Most of my recent projects have been in fullstack TypeScript, with React, NextJS, and Express.  

* 🌍  I'm based in Irvine, CA
* 🚀  I'm currently working on [Amplify Mobile](http://amplifymobile.io)
* 🧠  I'm currently learning React Testing Library, Playwright, and taking a course on Artificial Intelligence for fun 
* 📙  Next up, I'd like build a backend with Go
* 🤝  I'm open to collaborating on open source projects
* 🧑‍🎓  I studied Computer Science + Business Administration
* 👍  Fun fact: I was a product manager for many years, most recently at Facebook/Instagram, before making the crazy decision to follow my passion for engineering!
